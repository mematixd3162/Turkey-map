# 🗺️ Turkey SVG Map (Separated & Optimized)

A highly optimized, responsive, and interactive SVG map of Turkey with individual province paths. Built using pure HTML5, CSS3, and Vanilla JavaScript. Clean structure, modular architecture, and zero external dependencies.

Türkiye'nin tüm illerini barındıran, yüksek optimizasyonlu, responsive ve interaktif SVG haritası. Saf HTML5, CSS3 ve Vanilla JS ile modüler yapıda geliştirildi. Dışarıdan hiçbir kütüphaneye bağımlılığı yoktur, tamamen temiz ve performans odaklıdır.

---

## 🔥 Features / Özellikler

*   **%100 Responsive:** Scales perfectly automatically inside any parent container or dashboard. / Her türlü panele, ekrana ve container yapısına otomatik tam uyum sağlar.
*   **Modular Architecture:** HTML, CSS, and JS are strictly separated for clean codebase maintenance. / Temiz ve sürdürülebilir bir kod yapısı için HTML, CSS ve JS tamamen ayrı dosyalardadır.
*   **Performance Focused:** Lightweight SVG paths with optimized coordinates to ensure smooth hover and click actions. / Kasma ve gecikme olmaması için optimize edilmiş hafif SVG koordinatları ve akıcı hover/click eventleri.
*   **Data Visualization Ready:** Easy to bind with any API or database for regional data mapping (e.g., weather, sales, statistics). / Bölgesel veri haritalama projeleri (hava durumu, satış takibi, istatistikler) için veri bağlamaya tamamen hazırdır.

---

## 📂 Project Structure / Proje Yapısı

```text
├── 📄 index.html      # Contains the main layout and the optimized SVG map
├── 🎨 style.css       # Clean layout styling, responsive layout, and dark/light hover effects
└── ⚡ script.js      # Interactive event listeners (click & hover handlers) for provinces
🛠️ Setup & Integration / Kurulum ve Entegrasyon
To integrate this map into your web application / Haritayı projenize dahil etmek için:

Clone or download this repository. / Repoyu klonlayın veya indirin.

Link style.css inside your <head> tag. / CSS dosyasını <head> içine bağlayın.

Place the SVG structure from index.html wherever you want the map to render. / HTML içindeki SVG yapısını haritanın görünmesini istediğiniz yere gömün.

Include script.js right before the closing </body> tag. / JavaScript dosyasını </body> kapanışından hemen önce çağırın.
